# ChatClone
# Author: Minh Nguyen
# MERN Stack Chat App with Socket.IO

### Prerequisites
- Node.js
- React

### Installation

1. Set up .env
Create a .env file with your MONGO_DB_URI and SECRET_KEY

2. Install frontend dependencies:
```bash
cd frontend
npm install
npm run build
```

3. Running the Application
```bash
cd ..
npm install
npm run server
```
